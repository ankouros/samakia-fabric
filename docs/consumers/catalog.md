# Consumer Catalog

This is the **front door** for platform consumers. It is contract-only and read-only by default.

## Consumer types

- Kubernetes
- Database
- Message Queue
- Cache

For each consumer, choose a variant:

- **ready** (default): substrate only; no service deployed
- **enabled** (opt-in): service exists; Fabric enforces compliance against the contract
