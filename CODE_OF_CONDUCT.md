# Code of Conduct — Samakia Fabric

## Our Pledge

We as contributors and maintainers of **Samakia Fabric** pledge to make participation in this project
a respectful, professional, and harassment-free experience for everyone.

This project is built for serious infrastructure work.
Professional behavior is not optional.

---

## Our Standards

Examples of behavior that contributes to a positive environment include:

- Being respectful and considerate
- Focusing on technical merit and clarity
- Accepting constructive criticism gracefully
- Communicating clearly and professionally
- Respecting differing viewpoints and experiences

Examples of unacceptable behavior include:

- Harassment, discrimination, or hostility
- Personal attacks or insults
- Trolling, baiting, or deliberately disruptive behavior
- Publishing private information without consent
- Demeaning language related to personal characteristics
- Persistent off-topic or bad-faith arguments

Infrastructure work requires trust. Behavior that erodes trust is unacceptable.

---

## Scope

This Code of Conduct applies to:

- All project spaces (GitHub issues, pull requests, discussions)
- All community interactions related to Samakia Fabric
- Both human contributors and AI-assisted contributions

This applies regardless of role, contribution size, or seniority.

---

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

They have the right to take any action deemed appropriate, including but not limited to:

- Requesting clarification or correction
- Issuing warnings
- Temporarily or permanently banning contributors
- Closing or rejecting issues or pull requests

Decisions will be made with the goal of protecting the project and its contributors.

---

## Reporting

If you experience or witness behavior that violates this Code of Conduct:

- Do **not** escalate publicly
- Do **not** engage in arguments

Instead, report the issue privately to the project maintainer.

Reports should include:
- A description of the incident
- Relevant context or links
- Any available evidence

All reports will be handled discreetly and respectfully.

---

## Enforcement Guidelines

Maintainers will follow these principles when enforcing this Code of Conduct:

1. **Clarity** — Explain why behavior is unacceptable
2. **Proportionality** — Match response to severity
3. **Consistency** — Apply rules uniformly
4. **Project Safety** — Prioritize long-term health of the project

This is not a court of law.
It is an operational environment.
